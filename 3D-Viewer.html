Your Exportable HTML Code

Once you upload your .glb file to your web server (e.g. your hosting, WordPress Media library, Shopify file vault), copy this exact code to show it: 

<!-- Load Google's 3D Engine Component -->
<script type="module" src="https://ajax.googleapis.com/ajax/libs/model-viewer/3.4.0/model-viewer.min.js"></script>

<!-- The 3D Viewer Element -->
<model-viewer 
  src="your-file-name.glb" 
  camera-controls 
  auto-rotate 
  shadow-intensity="1" 
  ar 
  style="width: 100%; height: 500px;">
</model-viewer>
Crucial Reality Checks
  • Ensure file extension is strictly lowercase .glb
  • Keep file size under **5MB** so customers on mobile loading speeds don't bounce
  • Websites block 3D assets on standard http. Your site **must** have an https:// certificate (SSL) for it to load
